Place your project images here.

Recommended format:
- File format: .jpg or .png
- Size: 1200x600 px (2:1 ratio) or similar
- File naming: projectname-preview.jpg

Example:
- professional-project-placeholder.jpg
- personal-project-placeholder.jpg
- gamejam-placeholder.jpg

To use an image in your project:
Add `featured_image = '/images/projects/your-image.jpg'` in the project's frontmatter.
