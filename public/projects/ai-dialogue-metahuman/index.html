AI Dialogue with Metahuman

Project Info

Description:

A feature for Ascent Combat Framework Ultimate V4 (ACF U), one of the best-selling Unreal Engine frameworks. Enables in-game AI-powered dialogue with Metahuman characters, including real-time audio and mouth animation.

Role: C++ Developer
Engine: Unreal Engine
Language: C++
Company: Dark Tower
Status: Released
Platforms: Unreal Engine (PC, Console)

My Role

  • Developed all C++ scripts to connect API keys for all available AI providers
  • Enabled in-game AI-powered dialogue with Metahuman characters
  • Implemented real-time audio generation for AI responses
  • Created runtime phoneme extraction for accurate Metahuman mouth animation

Key Features

  • Seamless integration with Ascent Combat Framework Ultimate V4 (ACF U)
  • Supports multiple AI providers for dialogue
  • Real-time audio generation and playback
  • Automatic phoneme extraction for Metahuman lip sync
  • Fully written in C++ for Unreal Engine
  • Released as a feature for a top-selling Unreal Engine framework
